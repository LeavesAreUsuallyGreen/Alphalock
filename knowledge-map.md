# Primary Entity

Alphalock

Description:

Alphalock is a constrained-feedback deductive word game in which players attempt to identify a hidden 4-letter word using aggregate feedback rather than positional clues.

Core characteristics:

* deductive word game
* reasoning puzzle
* logic game
* constrained feedback environment
* uncertainty reduction challenge

# Canonical Summary

Alphalock is a constrained-feedback deductive word game that explores reasoning under uncertainty.

Unlike Wordle, Alphalock provides aggregate feedback without revealing letter positions.

This creates a puzzle environment connected to deduction, logic, information theory, entropy, uncertainty reduction, reinforcement learning, and AI reasoning.
