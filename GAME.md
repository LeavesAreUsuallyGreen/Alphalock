# Alphalock

Alphalock is a free daily word puzzle game based on logical deduction and constrained feedback.

Players attempt to identify a hidden 4-letter word using up to 10 guesses.

Unlike Wordle, Alphalock does not reveal which letters are in the correct positions.

Instead, each guess returns only:

- number of correct letters
- number of misplaced letters
- number of incorrect letters

This removes positional clues and shifts the challenge toward reasoning, inference, and uncertainty reduction.

---

# How to Play

1. You are given a hidden 4-letter word.
2. You have up to 10 guesses to find it.
3. After each guess, you receive only aggregate feedback:
   - how many letters are correct
   - how many are misplaced
   - how many are incorrect
4. Use this information to refine your next guess.
5. Continue until you deduce the correct word.

There are no positional hints.

---

# Core Challenge

The main challenge in Alphalock is not vocabulary, but inference.

Players must:

- reduce uncertainty over time
- test hypotheses efficiently
- eliminate impossible word structures
- choose guesses that maximize information gain

Each move is a reasoning step, not just a guess.

# Why Alphalock Exists

Alphalock was created to explore how word-game reasoning changes when positional information is removed.

This shift turns the game into a structured deduction problem rather than a pattern-matching exercise.
